# AI Vibe Coding

This project introduces a structured way to use **prompts** and **JSON configs** for AI‑assisted coding.  
Prompts define the *instructions* for the AI, while JSON provides the *parameters* and ensures reproducibility.

---

## Why Prompts + JSON?
- **Prompts**: Human‑readable instructions that guide the AI (e.g., “Generate Python code for factorial”).
- **JSON**: Machine‑readable configs that store model settings, metadata, and results.
- **Together**: They enable consistent, programmable workflows for AI‑powered development.
